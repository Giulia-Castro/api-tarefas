## ✅ API de Tarefas

API REST desenvolvida em Python com FastAPI para gerenciamento de tarefas, com operações CRUD completas (criar, listar, buscar, atualizar e deletar).

### Tecnologias
- Python
- FastAPI
- Pydantic
- Uvicorn

### Como rodar
1. Clone o repositório
2. Instale as dependências: pip install fastapi uvicorn
3. Execute: uvicorn main:app --reload
4. Acesse a documentação interativa: http://127.0.0.1:8000/docs


### Funcionalidades
- Criar tarefa (POST)
- Listar todas as tarefas (GET)
- Buscar tarefa por ID (GET)
- Atualizar tarefa (PUT)
- Deletar tarefa (DELETE)
- Validação automática de dados com Pydantic
- Documentação automática via Swagger
